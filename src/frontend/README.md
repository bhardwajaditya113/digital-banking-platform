# Banking Platform Frontend

React frontend for the Digital Banking Platform built with TypeScript, Redux Toolkit, and Bootstrap.

## Features

- User authentication (Login/Register)
- Account management
- Transaction history
- Money transfers
- Responsive UI with Bootstrap

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Running

```bash
npm start
```

The app will open at http://localhost:3000

### Building for Production

```bash
npm run build
```

## Environment Variables

Create a `.env` file:

```
REACT_APP_API_URL=http://localhost:8000
```
